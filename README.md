# LabVIEW-x-halftone

這個VI可以將bmp圖片或是jpg圖片轉成 半色調halftone的圖片

原圖:

轉換後:

![image](https://user-images.githubusercontent.com/103232200/162353918-c21ab96b-afbe-467c-928e-d6cd0cd488aa.png)

推薦一開始使用設定在下面這樣:
